Psc-Package Guide
============================

This is a guide for the tool `Psc-Package <https://github.com/purescript/psc-package>`_, which is a tool for managing PureScript dependencies via Git. It can be used directly or by external tools like `Pulp <https://github.com/purescript-contrib/pulp>`_.

.. note:: If there is a topic you would like more help with that is not in this guide, open a issue in the Github repo for it to request it.
